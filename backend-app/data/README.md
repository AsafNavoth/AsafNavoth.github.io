# Jamdict database

Place `jamdict.db` here for Japanese dictionary lookups.

**Download:** https://drive.google.com/drive/u/1/folders/1z4zF9ImZlNeTZZplflvvnpZfJp3WVLPk  
Extract `jamdict.db` from `jamdict-0.1a7.tar.xz`.

**Override path:** Set `JAMDICT_DB_PATH` env var (e.g. `/app/data/jamdict.db` on a server).
